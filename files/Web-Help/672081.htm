Split a package item into multiple package items

  1. Navigate to the required packing list number.
  2. Click to select the package which contains the item that you want to split.
  3. Click the Package Items tab.
  4. Select the tagged item that you want to split.
  5. Click Split Tag and then enter the number of items into which you want to split the selected item.
  6. Click Ok.
  7. Click Save.
