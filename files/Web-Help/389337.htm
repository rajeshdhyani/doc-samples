Add a message

  1. In the Message screen, click Add Message .
  2. Enter a relevant message code in the Message Code cell.

    Messages in Smart Materials start with the prefix MAR- or MAG-, followed by a five-digit number, for example, MAR-12345 or MAG-12345.

  3. Select the message type from the Type list.
  4. Click Expand to add the details of the message.
  5. In the Message Text box, type the message text.
  6. If there is a cause, action and hint for the message, type them in their respective boxes.
  7. Click Save.
