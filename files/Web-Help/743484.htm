Add a truck for a shipment

  1. Click the Shipment Details tab for the shipment for which you want to add a truck.
  2. Click Add Truck under Truck Details.

    • This option is available only for the shipments with Truck as Means of Conveyance.
    • If ATA of a shipment is entered, you cannot create new truck for that shipment.
  3. Select the packing lists or packages to be shipped by the truck and click Next.
  4. Type the Truck Number and the Shipping Document Number in the respective boxes.
  5. Enter the ETD and ETA in the respective boxes.
  6. Enter the name of the contact person for the truck in the Contact Person box.
  7. Enter contact person's phone number in Contact Phone Number box.
  8. If required, enter any additional information about the contact person in Other Reference box.
  9. Click Save.

    • You can click a truck number to view the assigned packages and make changes if required.
    • You can select a truck and click Delete truck to delete it. However, if the ATD of a truck is entered, you cannot delete it.
