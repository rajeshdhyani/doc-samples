View and edit associated requisition line items

If just one requisition line item is assigned to the agreement line item, you can update the quantity in the Line Items grid. If more than one requisition line is assigned, you must use this Requisition Line Items tab for the update, because the quantities finally purchased for requisition line items must be tracked to determine if a requisition has been purchased completely.

  1. Click Manage in the Other Details cell.
  2. Type the new quantity in the Quantity cell.
  3. Click Save.
