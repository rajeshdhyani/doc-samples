Delete a role

When can I delete a role?

You can only delete a role if it is not linked to any user. If a role is linked to users, you must first remove the role of these users to be able to delete the role. For information about how to remove a role that is linked to a user, see User Securities for Projects in the Smart Materials Administrator's Guide.

  1. From the Master Data navigation panel, click Administration > Role Management.
  2. Select a role, and click Delete Role .
