Sign the agreement

The signature can only be applied to the highest agreement supplement after approval. You can either manually sign the agreement or via electronic signature (e-signature).

If project default ZP_SMINCPO is set to N (default), the Signature panel is only enabled for subcontract agreements, except for notices to proceed. If ZP_SMINCPO is set to Y, this panel is available also for purchase order agreements.

  1. Click Signature on the category panel.
  2. Below Contractor, select either Manual to record a manual signature of the agreement, or E-Signature to perform an electronic signature.
  3. In the Signed By box, type the name of the signer.
  4. In the Title box, type the signer's title.
  5. If you apply a manual signature, you can update the signature date. Click Signature Date and pick a date.
  6. Click Save.
  7. In case of an e-signature a dialog appears where you must do the following:
    1. Read the displayed certification statements and select I agree to the above statements.
    2. If a secret question was defined, type the answer to your secret question.

      Click Forgot your answer? to receive an email with the answer. The email is sent to the address you've defined.

    3. Click Submit.

If your subcontractor signs the agreement in the Portal, his signature is displayed under Subcontractor on the right. If you want to enter the subcontractor's signature manually, proceed as follows:

  1. Select Manual.
  2. Type the name of the signer in the Signed By box.
  3. Type the signer's title in the Title box.
  4. Click Signature Date and pick the signature date.
  5. Click Save.
