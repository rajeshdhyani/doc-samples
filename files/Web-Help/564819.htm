Add new values for administrative master data items

  1. In the Master Data navigation panel, select the required master data item.
  2. Click Add , a new row for the data item gets added.
  3. (Optional) If the row has a project and a discipline list, select the required option from each list.
  4. Type the value for the new master data item. You can also copy or paste the data displayed in a grid layout; right-click the grid, and then select the required option.

    The details must be unique in the database.

  5. (Optional) Type a description and short description for the task in the respective boxes.
  6. Click Save.
