Edit allowances

To track allowances for a subcontract, you can define allowance rates. Allowance is an amount allocated to a specific item of work as part of an overall contract.

  1. Click Add Allowance .
  2. Click Rate Type and select a type.

    All types defined for Materials and Other groups are available.

  3. Optionally, enter the type description in the Short Description cell and in the Description box on the Info tab.
  4. Enter the allowance quantity in the Allowance Quantity cell.

    For all quantities exceeding this value, the rate entered here is applied.

  5. Click Quantity Unit and select the unit of the allowance quantity.
  6. Type the allowance rate in the Rate Value cell.
  7. If you want to change the agreement currency for this allowance, click Currency and select another currency.
