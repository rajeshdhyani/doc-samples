Remove a screen configuration

What is the difference between removing an assigned configuration and deleting a configuration?

Deleting a configuration deletes the configuration and all the projects assigned to that configuration. Removing an assignment removes the project associated with the configuration.

To remove an assigned project from a configuration, select the project, and then click the Remove Assigned Project icon.

To remove an assigned configuration, select a configuration on the Screen Configuration page, and click Remove Assigned Configuration icon.

At any point of time you can again assign a project to the same configuration.
