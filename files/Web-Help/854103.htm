Edit events

The event name, short description, and description cannot be changed.

  1. Click the expediting or inspection item to view the assigned events.
  2. To change the display order of an event, update the value in the Seq cell.
  3. To set or update the planned, the forecasted, or the actual date of an event, click and select a date.
  4. To mark a critical event select Critical.
  5. Optionally, type a comment on the event in the Comment box on the Description tab.
  6. To add further events, do the following:
    • Click Add Event .
    • Type a sequence number in the Seq cell.
    • Click Event and select an event.
    • Update the other values as described above.
  7. Click Save.
