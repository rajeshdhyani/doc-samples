Delete events

To delete an item including all assigned events, do the following:

  1. Select the expediting item or inspection item you want to delete.
  2. Click Delete Expediting/Inspection Item .
  3. Click Yes to confirm the message.
  4. Click Save.

To delete single events, do the following:

  1. Click the expediting or inspection item to view the assigned events.
  2. Select the event(s) you want to delete.

    Use the CTRL or SHIFT key to select multiple events.

  3. Click Delete Event .
  4. Click Save.
