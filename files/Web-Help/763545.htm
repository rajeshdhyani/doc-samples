View and edit agreement financials

  1. Click the Agreements task on the dashboard.
  2. Click an agreement number.
  3. Click Financials on the category panel.
  4. Click Other Costs to Edit agreement other costs.
  5. Click Total Costs to View and edit agreement total costs.
  6. Click Progress Payment Events to Edit progress payment events.

    If the agreement payment type is set to Milestone Progress, this option is only available if progress payment level is set to Header.

  7. Click Agreement Payment Sets to Define agreement payment sets.

    This option is only available if agreement payment type is set to Milestone Progress and payment level is set to Line Item.

  8. Click Forecast Payments to View forecast payments.
  9. Click Schedule of Values to Edit schedule of values.

    If project default ZP_SMINCPO is set to N (default), this option is only available for subcontract agreements.

  10. Click Guarantees to Assign guarantees to the agreement.
