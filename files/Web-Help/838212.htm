Define question sets

For technical and commercial questions you can define question sets, which can be attached to requisitions and inquiries instead of attaching single questions.

  1. Click Add Question Set .
  2. Type the set name in the Name box.
  3. If you want to copy an existing set, select the check box and a set from the list, and click Ok.
  4. Type the set description in the Short Description and Description boxes.
  5. Click Add Question .
  6. If needed, change the sequence number in the Seq box.
  7. Click Question and select a question.
  8. To add further questions, repeat the last to steps.
  9. Click Save.
