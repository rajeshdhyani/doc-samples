Edit detail milestones

If detail milestones exist for a milestone, the milestone appears as a link.

  1. Click the milestone.

    The Detail Milestones dialog box opens.

  2. Select the detail milestone you want to edit.

    Depending on the milestone type (requisition, inquiry, agreement, expediting), the displayed data may vary.

  3. Update the Weight Factor value if necessary.
  4. Click Save.
