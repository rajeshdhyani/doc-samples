Edit variation rates

To track price changes for an agreement, you can define variation rates.

  1. Click Add Variation Rate .
  2. Click Rate Type and select a type.

    All types defined for Materials and Other groups are available.

  3. Optionally, enter the type description in the Short Description cell and in the Description box on the Info tab.
  4. Click Quantity Unit and select the unit of the variation quantity.
  5. Type the variation rate in the Rate Value cell.
  6. If you want to change the agreement currency for this variation rate, click Currency and select another currency.
