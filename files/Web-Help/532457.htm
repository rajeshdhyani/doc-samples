View and manage notifications in the ribbon

When an invalid action is performed, a notification is displayed in a ribbon that appears at the top of the application window. The ribbon is displayed with a specific color code, icon, and a summary pane that shows the number of notifications next to each icon, as shown in the screen shot below:

  1. For multiple invalid actions, the software displays the following generic message:
    • Use the visual cue to navigate to the problem details.
  2. For a single invalid action, the software displays a message specific to the invalid action. For example:
    • Field cannot be empty
    • A negative value is not allowed
    • Username abc already exists
