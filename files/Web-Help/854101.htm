Add expediting events

You can link events to the agreement, to the requisition that is assigned to the agreement, and to the item shipments of the agreement.

  1. Click Expediting Events.
  2. Click Add Expediting Item .
  3. In the Define Expediting Scope dialog box, click Expediting Level and select a level.
  4. To define where to link the event, do one of the following:
    • Select Agreement, and click OK.
    • Select Engineering Requisition, click Next, select the requisition(s) you want to assign, and click OK.
    • Select Item Shipment, click Next, select the item shipment(s) you want to assign, and click OK.
  5. Click Save.

    All expediting events defined for the selected level are added to the selected item.
