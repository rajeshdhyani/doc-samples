Translate messages

You can translate messages from one language to another.

  1. In the To list, select the language you want the message to be translated to, as shown in the following screen shot:

    The language that you select in the Data Language box appears in the To list by default. See Configure the user settings for more details.

  2. In the From list, select the language used in the message.
  3. Click Expand to view the message details.
  4. Type or paste the translated message in the Message Text, Cause, Action, and Hint boxes.

    The source language text appears as a reference below each box, as shown in the following screen shot:

  5. Click Save.
