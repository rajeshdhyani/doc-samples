Refresh milestones

This refresh functionality allows you to easily add all milestone labels to the selected supplement plan, which are part of the milestone set that is assigned to the associated planned requisition, but that are currently not included in the selected supplement plan. Additionally, this functionality can be used to update the milestones when the milestone set definition has been changed by adding or removing milestones. Using this option is more convenient than adding the milestone labels manually one by one.

  • Click Refresh .

    The milestones will be adjusted to the assigned milestone set.
