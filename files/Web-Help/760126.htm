Assign heat numbers

The heat number is a sign for the heat that allows the tracing of the material analysis of the marked item.

  1. Select an item shipment.
  2. Click the Heat Numbers tab.
  3. Click Create Heat Number .
  4. Type the heat number in the Heat Number cell.
  5. Enter the heat number quantity in the Quantity cell.

    • The total quantity of all heat numbers assigned to an item shipment must not exceed the item ship quantity.
    • Heat number and quantity are mandatory and must be entered, all other values described below are optional.
  6. Type the certificate number assigned to the heat number in the Certificate Number cell.
  7. Type the name of the manufacturer who produces the item with the assigned heat number in the Manufacturer cell.
  8. Type the text marked on the item with the assigned heat number in the Mark cell.
  9. Type the identification or reference name for the item used by the supplier in the Vendor Code cell.
  10. Type the location of the file that contains the certificate(s) in the File Location cell.
  11. Some material consists of two parts and therefore requires a second heat number. Type this second number in the Heat Number 2 cell.
  12. Type the certificate number assigned to the second heat number in the Certificate Number 2 cell.
  13. Type the name of the manufacturer who produces the item with the assigned heat number 2 in the Manufacturer 2 cell.
  14. Type the text marked on the item with the assigned heat number 2 in the Mark 2 cell.
  15. Click Save.
  16. To assign properties to the heat number, click Manage Properties , and see Manage properties.

To delete an assigned heat number, select the heat number and click Delete Heat Number .
