Define decline reasons

Decline reasons are used in the Portal. If a bidder does not want provide a bid for an inquiry, he selects the I will not bid option and a decline reason.

  1. In the Master Data navigation panel, select Procurement > Decline Reasons.
  2. Click Add Decline Reason .
  3. Type a unique sequence number in the Seq cell.

    This sequence number controls the display order sequence of the decline reasons in the Portal.

  4. Type the reason in the Decline Reason cell.

    To define reasons in another language, login with this language.

  5. Click Save.
