Create suppliers

You can only create suppliers or subcontractors if the company already exists.

  1. Click the Suppliers task on the dashboard, or select Suppliers from the Tasks list.
  2. Click Create Supplier .
  3. Type the supplier's name in the Supplier box.
  4. Click Company Name and select the name of the company to which you want assign the supplier.
  5. Click Company Type and select the supplier's company type.
  6. Type a value in the Sales Limit box.
  7. Click Preferred Currencies and select the supplier's preferred currency.
  8. Click Save.

    All other fields are optional and can be filled later. For more information see View and edit the supplier summary.
