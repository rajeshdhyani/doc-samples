Translate questions

You can translate questions to use them in other languages.

  1. Click a question.
  2. Click the Translation tab.
  3. Select the language from the Translate to list.
  4. Type the question translation in the Question box.
  5. If the answer type is LOV or Multiple Answers, and you have defined manual select list values, translate these values by entering the translation in the last column.
  6. Click Save.
