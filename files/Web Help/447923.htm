View and edit agreement details

  1. Click the Agreements task on the dashboard.
  2. Click an agreement number to view and edit the following details:
  • Edit agreement header information
  • Assign and edit agreement properties
  • Edit supplier information
  • Edit responsibilities
  • Edit financial details
  • Edit delivery details
  • Edit agreement status
  • Edit subcontract status
  • Edit notice to proceed details
  • Sign the agreement
  • Edit post agreement information
