Filter the tree

  1. Click to open the filter dialog box.

  2. Select check boxes for the items that should appear in the tree.
  3. Select an operator from the Show rows with value that drop-down list.
  4. Enter a string in the text field.
  5. Click to make the filter case sensitive.

    To add an additional search criterion, select And or Or from the next list. If you select And, both criteria must match to display a record. If you select Or, one of the two criteria must match. Then select an operator from the next drop-down list, and enter a string in the text field.

  6. Click Filter to apply the filter.
