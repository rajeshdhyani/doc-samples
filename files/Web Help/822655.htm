Terms and Acronyms

Use the following topics to learn more about key terms and acronyms in Smart Materials.

Terms

Term

Meaning

Bill of Materials

This is the list of everything needed to deliver the project.

Material Receiving Report (MRR)

This records the materials that have been received from a supplier and other related details.

Milestone

This is an event with a date that requires tracking.

Milestone set

This is a group of related milestones.

Over, Short, and Damaged (OSD) report

This contains details of the item that have been over-shipped, short-shipped, or received in an unsatisfactory or damaged condition.

Requisition

This is the list of what need to be bought. Also known as a summarized BOM.
