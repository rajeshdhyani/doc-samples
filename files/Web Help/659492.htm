Configure the display of the tasks on homepage

The Dashboard Configurator allows you to arrange the tasks on the dashboard. However you can only arrange the tasks for which you have access.

  1. Right-click the dashboard, and click Dashboard Configurator.
  2. Select the number of columns you need.
  3. Select a task in Available Widgets, and click Add.

    To remove a task, select a task from the Selected Widgets, and click Remove.

  4. Click Up or Down to reorder tasks under the required column.

    You can drag a task on the dashboard to the required column.

  5. Click OK to save the configurations.
