View and edit message details

To get a list of messages that do not have all the details entered, do the following:

  1. In the Translate group, click Show messages with incomplete details.

    Messages that do not have details in any of the boxes (Cause, Action, Hint) appear.

  2. Click Expand to view the message details.
  3. Type or edit the message text in the missing entry boxes.

To view all the messages, clear the Show messages with incomplete details check box.
