Create a planned tag

  1. Click the Planning task on the dashboard, or select Planning from the Tasks list.
  2. Click Create Planning Item to create a new planning item.
  3. Select Tag from the Planning Item Type list.
  4. Type or select a tag number from the list in the Tag Number box.
  5. If you create a planned tag in all-project mode, select a project from the Project drop-down list.
  6. Click Milestone Set and select a milestone set.
  7. (Optional) Click ROS Date and select the required on site date.
  8. (Optional) Type the client requisition number in the Client Req Number box.
  9. (Optional) Click Account Code and select an account code.
  10. (Optional) Type the client agreement number in the corresponding box.
  11. (Optional) Click Planning Group and select a planning group.
  12. (Optional) Type the budget values in the Labor Budget 1 and Labor Budget 2 boxes.
  13. (Optional) Type a comment on the planning item in the Comments box.
  14. Click Save.
