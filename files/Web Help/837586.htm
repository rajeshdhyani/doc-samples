Attach documents to questions

  1. Click a question.
  2. Click the Attachments tab.
  3. Select the file you want to upload, drag and drop it into the Attachments tab.
  4. Alternatively, do the following:
    1. Click Attach File .
    2. Select a file.
    3. Click Open.
  5. Click Save.

  • To view an attached file, click the file name.
  • To delete an attached file, click X next to the date.
