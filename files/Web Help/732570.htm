Manage requests for information

You can create requests for information (RFI) to ask questions to your suppliers to get clarification regarding the registration process.

Edit a request for information

  1. Click the Suppliers task on the dashboard, or select Suppliers from the Tasks list.
  2. Select Request for Information from the list on top of the page.
  3. Select a company in the tree, and click to view the existing RFIs for this company.
  4. Select an RFI to view the details on the right.

    - the answer on a question of this RFI is pending

    - the supplier has answered all your questions

  5. To send a question to the supplier, type the question and click Save.

    An email is sent to the supplier.

Create a request for information

  1. Select the company for which you want create an RFI.
  2. Click Add RFI .
  3. Type the RFI subject in the Subject box.
  4. Type a question in the Question box.
  5. Click Save.

    An email is sent to the supplier

Delete a request for information

  1. Select the RFI you want to delete.
  2. Click Delete RFI .
