Define conditional questions

If the answer type is Y/N, you can define conditional questions which are displayed depending on the given answer.

  1. Select the Y/N question.
  2. Click the Conditional Questions tab.

    You can define conditional questions for answer Yes and No.

  3. Click Add Question .
  4. Click Question and select a question.

    All questions are available for selection that have been defined for the same combination of company type, level 1, level 2, level 3, level 4 like the master question, and that do not have conditional questions assigned.

  5. Repeat the previous steps to add more questions, and save.
