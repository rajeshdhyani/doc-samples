Create agreements

You can create agreements based on quote summaries, requisitions, corporate warehouses, and free-issue material (FIM) warehouses. Additionally, you can create agreements manually.

  1. Click the Agreements task on the dashboard.
  2. Click Add Agreement .

    This button is enabled only for users with the ASSIGN BUYER or BUYER privilege.

  3. In the Create New Agreement dialog box, select one of the following options to continue:
    • Create the agreement based on a quote summary
    • Create the agreement based on an engineering requisition
    • Create the agreement based on a corporate warehouse
    • Create the agreement based on a FIM warehouse
    • Create a manual agreement
