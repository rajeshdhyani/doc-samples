Create a role

Why do I need to create a role?

You will need to create a role so that you can control the display of various options on the Smart Material screens based on a user role.

  1. From the Master Data navigation panel, click Administration > Role Management.
  2. Click Add Role .
  3. In the Role box, type a role name.
  4. (optional) In the Description box, type a description of the role.
  5. Select the Hidden check box for the screens you want to exclude from the role.

    All child elements of the selected screen are also excluded.

  6. Select the Read-Only check box for the screens you want to include in the role in read-only mode.

    All child elements of the selected screen are also set to read-only.

    All screens that do not have Hidden or Read-Only selected are included in the role with full access.

  7. Click Save.
  8. Click Next.
  9. If you want to restrict the access rights for a screen on a deeper level, expand the screen and do the following:
    • Select the child element for which you want to apply the restriction.
    • Select the Hidden check box to exclude the child element ( for example, navigation tree, panel, tab, dialog box) from the role.
    • Alternatively, select the Read-Only check box to restrict the access right for the selected child element.
    • Click Save.
  10. Repeat the previous step for all screens and elements you want to further restrict the access rights.

  • The linking of a role to a user is managed in Smart Materials Classic.
  • For more information, see Roles in the Smart Materials Administrator's Guide and see User Securities for Projects in the Smart Materials Administrator's Guide.

At any point, you can update the role.
