Add a price category

  1. From the EML Management page, click the EML. The EML related information is displayed on the EML Details tab.
  2. Click the Price Codes tab.
  3. Click Tree View to view the price code categories in a tree structure.
  4. Click Add .
  5. Select Add Category if you want to add the category at the same level as the currently selected category. Select Sub Category if you want to make the new category a child category.
  6. Type the category code and name in the Category Code and Category Name boxes.
  7. (Optional) Select the price codes that you want to be included in this category.
  8. Click Save.

    To delete a category, select the category and click Delete .
