Specify containers for preliminary packages or packages

Who all can own or provide a container?

A container can be owned or provided by the shipping company, supplier, or freight forwarder. You assign the owner of a container in the Container Provider box. However, the freight forwarder option is only available if the PPL or the packing list has an EML number assigned.

  1. Navigate to the required PPL or packing list.
  2. If all the packages in the PPL/packing list share the same container, perform the following steps.
    1. On the Details tab, select PPL Level or Packing List Level from the Provide Container Characteristics at list.
    2. Click Manage Container Characteristics.
    3. Click Specify Container.
    4. In the Container Provider box, select the provider / owner of the container.
    5. In the Container Number box, enter a container number.
    6. In the Container Type box, enter the container type.
    7. In the Container Dimension box, enter the container dimensions.
    8. Select the Out of Gauge check box, if the packages extend beyond the container.
    9. Click Save.
  3. If all the packages in the PPL/packing list do not share the same container, perform the following steps.
    1. On the Details tab, select Preliminary Packages Level or Packages Level from the Provide Container Characteristics at list.
    2. Select a package under the PPL/packing list.
    3. On the Details tab, enter the Container Provider, Container Number, Container Type, and Container Dimension in the corresponding boxes under Container Characteristics.
    4. Select the Out of Gauge check box, if the package extends beyond the container.
    5. Click Save.
    6. Repeat the above steps for all the packages in the PPL/packing list.
