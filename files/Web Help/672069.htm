Create a preliminary package

  1. Navigate to the required PPL..
  2. Click Add Preliminary Package.
  3. Type the package number in the Package Number box.
  4. Enter the package type in the Type box.
  5. Type the number of preliminary packages that you want to create, in the Number of Packages box.

    Select the Lot Package check box, if the package might have more quantities of some items later. Mention such items in the Description box. Lot packages are not considered for HLO calculation.

  6. If required, type a short description of the package in the Short Description box.
  7. If required, type a description of the package in the Description box.
  8. Do one of the following to provide the volume of the package.
    • Type the dimensions of the package in the Length, Width, and Height boxes. The volume is automatically calculated based on the dimension values.
    • Type the volume of the package in the Volume box.

      The Volume box is only editable when all the dimension boxes values are zero.

  9. Type the gross weight of the package in the Gross Weight box.

    • The value in the Total Freight Ton box is automatically calculated from the Total Volume and Total Gross Weight value.
    • If required, type a value in the Net Weight box.
    • You can select the unit of measure from Weight Unit list.
  10. If required, enter a code in the Site Storage Code box.
  11. If required, enter stacking information about the package in the Stackable Tier box.
  12. If required, type a comment in the Shipping Comments box.
  13. Click Save.

    You can also create a preliminary package by selecting an existing preliminary package and clicking Duplicate Package. That creates a new preliminary package containing the same information as the selected preliminary package. The package items are not copied when you duplicate a preliminary package.

    • The list of units linked to the unit type PKG_WEIGHT are used to assign weight units to packages or preliminary packages. However, only those units for which unit-to-unit conversion factors are set in A.10.12 are displayed (to the suppliers) in the Weight Unit list.
    • The list of units linked to the unit type DIMENSION are available in the Dimension Unit list for the suppliers to assign dimension units to packages or preliminary packages. You should assign units only from the unit groups - LENGTH, QUANTITY, and DIMENSION to the DIMENSION unit type. If the DIMENSION unit type is not available, you can create it in A10.12 screen.
