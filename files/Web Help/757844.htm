Add shipping invoices for a shipment

Do I need to own a shipment in order to add shipping invoice to it?

No, you can create shipping invoices even when you do not have the ownership of the shipment. However, to make any other changes in the shipment details, you need to have the ownership.

  1. Click the Custom Clearance tab for the required shipment.
  2. If you want to add import shipping invoices, click Import Customs Clearance. By default, Export Customs Clearance is selected.
  3. Select the packing lists for which you want to add a shipping invoice.
  4. Click Add Shipping Invoices .
  5. Click Add Shipping Invoices on the Shipping Invoice Details tab.
  6. If you want to create a new shipping invoice:
    1. Click New.
    2. Enter the required details and click Save.
  7. If you want to use an existing invoice:
    1. Click Existing.
    2. Select the shipping invoices that you want to use.
    3. Click Add.

    • Select the Assign to Freight Forwarder check box, if you want the shipping invoice to be available to the freight forwarder.
    • Select the Assign to Supplier check box, if you want the shipping invoice to be available to the supplier.
    • To delete a shipping invoice, select it and click Delete .
    • You can attach files on the Shipping Invoice Details tab, by selecting the appropriate shipping invoice and dragging the files under Attachments.
