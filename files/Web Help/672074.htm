Modify package item details

  1. Navigate to the required PPL or packing list.
  2. Click to select the package for which you want to modify the package item details.
  3. Click the Package Items tab.
  4. Click to select the package item that you want to modify. Item details are shown.
  5. If required, change the quantity in the Total Quantity box.
  6. If the package item includes hazardous material, select the Hazardous check box. You can provide more details about the hazard in the Hazard Details box under Description.
  7. If required, on the Description tab, type a comment in the Shipping Comments box.
  8. Click Save.
