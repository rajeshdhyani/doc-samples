Recalculate planned dates

  1. Enter a planned date for one milestone.

    • If you enter more than one planned date, the recalculation will consider only the value of the milestone with the lowest sequence. All other values will be overwritten.
    • For a two-phase planning, you must enter a date in the 1st phase (between first milestone and 1st pass milestone) and a date in the 2nd phase (between 2nd pass milestone and the last milestone).
  2. If the plan has changed from the original plan, update the values in the Days cell accordingly.
  3. Click Save.
  4. Click Update Planning .
  5. Click Recalculate Planned Dates.

    The planned dates are calculated based on the entered date(s) + span days + add days.

To delete the planned dates, click Update Planning > Clear Planned Dates.
